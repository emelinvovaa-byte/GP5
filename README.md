## Данные
Данные взяты с сайта https://youla.ru/
Датасет представляет из себя данные об объявлениях с продажей обуви на сайте 
Price - цена
Dcr - описание 
src - ссылка на картинку
Категория - категория(мужской гардероб)
Подкатегория - подкатегория(обувь)
Тип - тип(кроссовки)
Размер - размер
Состояние товара - состояние(новый/бу)
Бренд - бренд
Цвет - цвет
Сезон - сезон(лето, зима)
img_name - путь к картинке
category - категория созданная нами(domash)
Тип каблука 
Высота каблука
